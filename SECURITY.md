# Security Policy

## Supported versions

Only the latest release receives fixes.

## Reporting a vulnerability

Please do not open public issues for security problems. Report privately via
GitHub Security Advisories on this repository (preferred) or by email to
velorhqofficial@gmail.com. You can expect an acknowledgment within a week.

## Scope notes

Meridian is a research literature assistant, **not medical advice**, and is not
intended for deployment in safety-critical settings. Reports about answer
quality or hallucination belong in regular issues; this policy covers software
vulnerabilities (e.g., in the serving API added in later phases).
